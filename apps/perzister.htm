key notes

  • html5, javascript & css single page application
  • based on jQuery , Angular & responsive bootstrap front end frameworks | read guide to bootstrap
  • JSONP & cross domain ajax calls
  • ready for phonegap
  • include plugins : datatables, local live search, back strech, ...
  • jupiter software fetaures : operater login, searchbox look & feel, friendly import NetGen forms
  • printer friendly css
  • built-in multi language support
  • symbols first styling with font-awesome help
  • other : enter as tab, application button automatic fill
  • optional dark or white color shema css

  • nav bar fixed on top and one section visible
